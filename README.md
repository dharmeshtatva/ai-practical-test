# Node.js practical test for engineer.ai

## Steps to Start Application
 1. Clone this repository
 2. go to cloned folder
 3.  run `npm install`
 4. copy `.env.example` to `.env` and set MongoDB Connection URL
 5. run `npm run seed`
 6. run `npm start`

## SEED Data
 - run `npm run seed`
